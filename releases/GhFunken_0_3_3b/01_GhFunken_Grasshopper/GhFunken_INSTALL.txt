-----STEP#01 - GhPython Installation (ONLY for Rhino 5) -----
1 - If you do not have GhPython installed, download it from Food4Rhino: www.food4rhino.com/app/ghpython
2 - Copy the .gha file in the Grasshopper libraries folder (File -> Special folders -> Libraries)
3 - Right click on the copied file, go to properties and make sure the file is not blocked.
4 - Reload Grasshopper and under Math/Scripts you should have a new Python component.
-----

-----STEP#02 - GhFunken Library -----
1 - Unzip this folder
2 - Copy the "GhFunken" folder to Grasshopper Components folder (File -> Special folders -> Components)
3 - Restart Rhino
-----

-----STEP#03 - GhFunken User Objects -----
1 - Copy the content (just the content not the whole folder) of the userObjects folder into Grasshopper user objects folder (File -> Special folders -> User Obejcts)
2 - The Funken tab should appear in the Grasshopper toolbar
